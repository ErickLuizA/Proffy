## :rocket: Getting Started

1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/Proffy.git
```

2. Installation

``` shell
# Go into the repository
cd Proffy

# Go to the web folder
$ cd web

# Install dependencies
$ yarn install
```

3. Execution

```shell
# Start the web
$ yarn start
```
